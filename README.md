# Network-Port-Scan
Learn to discover open ports on devices in your local network to understand  network exposure.

For Security Concern We going to Use NAT Network.


First We have to Install Nmap on our Device for Now i am using Linux 
Nmap:- https://nmap.org/download

Then I use Nmap Command to Scan all Decices in the Network 
Command:- nmap -sS 192.168.153.130/24          -sS : TCP SYN scan

I use Nmap CheetSheet For Further Proccess 
CheetSheet:- https://www.stationx.net/nmap-cheat-sheet/

Then I Search For Vulnerability For Each Port by Service Name i got Some High , Critical , Medium Risk
